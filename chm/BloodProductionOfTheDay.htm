"Ручная" заготовка крови

Для того, чтобы внести информацию по ежедневной заготовке крови "ручным" способом необходимо выбрать пункт меню: Заготовка за день/"Ручная" заготовка крови

Рабочая область разбита на несколько областей, как показано на рисунке.

  1. В данной области указываем дату заготовки (как правило текущую дату); количество человек прошедших процедуру донации (указывается общее количество, даже если заготовка происходила в разные гемаконы); объем гемаконов, используемых при донациях (если в течении дня использовали два вида гемаконов при "ручном" заборе крови, то смотри описание 6й области); количество доз.

  2. Во втором блоке заносится информация по заготовке цельной крови (до фракционирования).

  3. В третьем блоке заполняется информация по компонентам, полученным в результате фракционирования заготовленной цельной крови.

  4. Ставим галочку в том случае, если при заготовке использовались 2 вида гемаконов. При этом в первом блоке появляется две дополнительные ячейки (объем 2го гемакона и количество заготовленных доз). Кроме этого появляется дополнительная область (копия 3го блока), в которую вносится информация по компонентам, полученным при фракционировании цельной крови заготовленной с помощью второго вида гемаконов.

  5. Ставим галочку в том случае, если заготовка крови происходила в выездных условия.

  6. Для того, чтобы активровать основные кнопки, необходимо нажать кнопку "Разблокировать".

    • Добавить - добавляется новая запись, которая срузу отображается в блоке 7.

    • Изменить - позволяет отредактировать существующую запись. Для этого необходимо выбрать интересующую запись в блоке 7 и нажать кнопку "Изменить". Для фиксации изменений необходимо воспользоваться кнопкой "Сохранить".

    • Удалить - позволяет удалить существующую запись. Для этого необходимо выбрать интересующую запись в блоке 7 и нажать кнопку "Удалить".

  7. В седьмом блоке отображаются существующие записи, содержащие информацию по заготовке за предыдущие даты.
