Автоматичечкий аферез

Для того, чтобы внести информацию по заготовке плазмы методом автоматического афереза необходимо выбрать пункт меню: Заготовка за день/Автоматический аферез

Рабочая область разбита на несколько областей, как показано на рисунке.

  1. В данной области указываем дату заготовки (как правило текущую дату); количество человек прошедших процедуру донации.

  2. Во втором блоке заносится информация по цельной крови переработанной в процессе заготовки плазмы.

  3. В третьем блоке заполняется информация по количеству полученной плазмы методом автоматического афереза.

  4. В четвертом блоке отображаются существующие записи, содержащие информацию по заготовке за предыдущие даты.

  5. Для того, чтобы активровать основные кнопки, необходимо нажать кнопку "Разблокировать".

    • Добавить - добавляется новая запись, которая срузу отображается в блоке 4.

    • Изменить - позволяет отредактировать существующую запись. Для этого необходимо выбрать интересующую запись в блоке 4 и нажать кнопку "Изменить". Для фиксации изменений необходимо воспользоваться кнопкой "Сохранить".

    • Удалить - позволяет удалить существующую запись. Для этого необходимо выбрать интересующую запись в блоке 4 и нажать кнопку "Удалить".
